# Kagevo — Design Prototype

Voice-first AI companion UI prototype. Single self-contained HTML file — no build step.

## Open

```sh
open index.html
```

## What's inside

- Interactive onboarding (custom kage or anime alter ego)
- Goals manager (daily/weekly, brutal edit block)
- Spotify-style share sheet
- P0 settings (notifications, crisis safety, privacy)
- Static gallery of all screens

## Stack

HTML · CSS · vanilla JS · Google Fonts (Inter, JetBrains Mono)
