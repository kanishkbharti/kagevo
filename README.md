# Kagevo — Design Prototype

Life-RPG UI prototype. **Final design:** `mvp.html` (manga ink · Seasons · Sensei · Guilds).

## Open

```sh
open mvp.html
```

Earlier iterations: `index.html`, `index2.html`, `anime.html`, `landing.html`.

## What's inside (mvp)

- Auth gate (sign in / register / OAuth)
- 8-screen onboarding → AI journey generation
- Journey dashboard (Top 3 MITs, side quests, focus, skip/victory states)
- Story tab (Season → Chapter → Quest → Boss)
- Sensei tab (voice-first, chat, weekly review)
- Character tab (level, attributes, titles, heatmap)
- Guilds (discover, leaderboard, weekly challenges)
- Journal & life arcs timeline
